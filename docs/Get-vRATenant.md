# Get-vRATenant

## SYNOPSIS
Retrieve vRA Tenants

## SYNTAX

```
Get-vRATenant [[-Id] <String[]>] [[-Limit] <String>]
```

## DESCRIPTION
Retrieve vRA Tenants.
Make sure to have permission to access all Tenant information

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-vRATenant
```

### -------------------------- EXAMPLE 2 --------------------------
```
Get-vRATenant -Id Tenant01
```
