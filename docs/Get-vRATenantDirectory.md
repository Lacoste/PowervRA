# Get-vRATenantDirectory

## SYNOPSIS
Retrieve vRA Tenant Directories

## SYNTAX

```
Get-vRATenantDirectory [-Id] <String[]> [[-Limit] <String>]
```

## DESCRIPTION
Retrieve vRA Tenant Directories

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-vRATenantDirectory -Id Tenant01
```

### -------------------------- EXAMPLE 2 --------------------------
```
Get-vRATenantDirectory -Id Tenant01,Tenant02
```
